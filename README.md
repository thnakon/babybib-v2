## ğŸ“¦ Installation & Setup

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/thnakon/scribehub.git
    cd scribehub
    ```

2.  **Environment Setup**
    ```bash
    cp .env.example .env
    # Configure your database and GEMINI_API_KEY
    ```

3.  **Backend Setup**
    ```bash
    composer install
    php artisan key:generate
    php artisan migrate
    ```

4.  **Frontend Setup**
    ```bash
    npm install
    npm run dev
    ```

5.  **Go Live**
    ```bash
    php artisan serve
    ```
